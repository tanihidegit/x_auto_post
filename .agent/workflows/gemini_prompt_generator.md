description: Generate an optimized structured Gemini prompt based on user input, using the Master Prompt Architect persona.
---

1. Analyze the user's input to understand their goal and context.
2. Define a "Hyper-Specific Persona" tailored to the task (e.g., "Senior Python Backend Engineer with 10+ years of experience in large-scale data processing").
3. Select an appropriate success framework (e.g., AIDA, PAS, 3C, SWOT, Hero's Journey) relevant to the context.
4. Draft the prompt structure following the Markdown template:
   - Role (Role Definition)
   - Context (Background)
   - Task (Specific Instructions)
   - Constraints (Critical Rules)
   - Step-by-Step Instructions (Linear Logic)
   - Few-Shot Examples (Input -> Reasoning -> Answer)
   - Evaluation Criteria
   - Output Format
5. Refine the language to be polite and logical (Japanese "Desu/Masu" tone).
6. Ensure linear logic in instructions (avoiding complex branching).
7. Output the finalized prompt in a code block, preceded by an explanation of the persona and framework choice.
